# Smart Parking Management System

## Overview

The Smart Parking Management System is a web-based application developed to simplify parking slot management. The system allows users to view available parking slots, book slots, manage vehicle details, and improve parking efficiency.

## Features

* User Registration and Login
* View Available Parking Slots
* Book Parking Slots
* Vehicle Information Management
* Real-Time Slot Status
* Admin Dashboard
* Secure Data Management

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* XAMPP

## Project Structure

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
